Mac+Hexo+GitHub建Blog问题集合

总结一下在建站过程中遇到的小问题，日常更新。

主题相关：

1 Hexo本地测试运行重启后页面空白，提示：WARN No layout: index.html?

解决办法：出现这种问题一般是因为主题安装包没有放在正确的位置，Blog根目录下的_config.yml无法正常获取主题，所以检查一下以下两点：

  1. 检查一下你是否把主题文件夹（例如主题名为icarus）正确放置在了你Blog的文件夹下面的themes文件夹中:

  1. 检查一下_config.yml文件中（是Blog总文件夹下的，不是themes文件夹下的）： 找到这一段，看看theme: icarus这里是不是你现在用的主题文件夹的名字。比如我在用icarus主题，这里就应该是icarus。

2 主页三栏，阅读模式两栏

  1. 将_config.yml文件复制两份，分别更改文件名为_config.page.yml和_config.post.yml。根据icarus主题的设定，文件_config.post.yml和_config.page.yml中的配置将覆盖默认配置文件_config.yml中的配置。因此，如果想要在阅读模式下分为两栏，更改_config.post.yml文件就可以了。

  2. 打开_config.yml中的配置，可以看到widgets的position属性有的在左有的在右，所以是三栏。

  3. 打开_config.post.yml中的配置，widgets全部在左，所以是两栏。
